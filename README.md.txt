# ATM Management System

## Overview
A secure ATM Management System built using Python, Tkinter, and MySQL.

## Features
- Deposit Money
- Withdraw Money
- Balance Checking
- PIN Authentication
- Change PIN
- Transaction History
- SHA256 PIN Hashing
- MySQL Database Integration

## Technologies Used
- Python
- Tkinter
- MySQL
- hashlib

## Database Features
- Stores account details securely
- Maintains transaction history
- PIN hashing using SHA256

## How to Run

1. Install Python
2. Install dependencies:

```bash
pip install mysql-connector-python
```

3. Configure MySQL database
4. Run:

```bash
python atm.py
```
